# POO2README del Proyecto: Encriptación y Desencriptación en Java

Este proyecto desarrolla un sistema sencillo para encriptar y desencriptar texto utilizando Java, permitiendo transformar un mensaje original a uno cifrado mediante desplazamiento de caracteres, así como recuperar el mensaje original aplicando el proceso inverso

Descripción del Proyecto

El programa permite ingresar una cadena de texto, procesarla a través de un algoritmo simple que modifica el código ASCII de cada carácter y generar una versión encriptada del mensaje, del mismo modo, permite revertir el proceso para obtener el texto original sin pérdida de información

Funcionalidades

Encriptación básica de texto

Desencriptación del texto cifrado

Visualización del mensaje original, encriptado y desencriptado

Código simple para aprendizaje y práctica
